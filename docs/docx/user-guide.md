## Начало работы

### Регистрация

Для использования всех возможностей GameFast необходимо создать учетную запись:

1. На главной странице нажмите кнопку **"Регистрация"** в правом верхнем углу.
2. Заполните форму регистрации:
   - **Имя пользователя** - придумайте уникальное имя пользователя
   - **Email** - укажите действующий адрес электронной почты
   - **Пароль** - придумайте надежный пароль (минимум 8 символов, содержащий буквы и цифры)
   - **Подтверждение пароля** - повторите введенный пароль
3. Прочитайте и примите условия пользовательского соглашения, установив соответствующий флажок.
4. Нажмите кнопку **"Зарегистрироваться"**.
5. На указанный email будет отправлено письмо с ссылкой для подтверждения регистрации.
6. Перейдите по ссылке из письма для активации аккаунта.

### Вход в систему

После регистрации вы можете войти в систему:

1. На главной странице нажмите кнопку **"Войти"** в правом верхнем углу.
2. Введите свой email и пароль.
3. Установите флажок **"Запомнить меня"**, если хотите оставаться в системе.
4. Нажмите кнопку **"Войти"**.

### Восстановление пароля

Если вы забыли пароль:

1. На странице входа нажмите ссылку **"Забыли пароль?"**.
2. Введите email, указанный при регистрации.
3. Нажмите кнопку **"Восстановить пароль"**.
4. На указанный email будет отправлено письмо с инструкциями по сбросу пароля.
5. Перейдите по ссылке из письма и введите новый пароль.

## Навигация по сайту

### Главная страница

Главная страница GameFast содержит:

- **Верхнее меню** - навигация по основным разделам сайта
- **Слайдер с акциями** - актуальные предложения и скидки
- **Популярные товары** - товары, пользующиеся наибольшим спросом
- **Новинки** - недавно добавленные товары
- **Категории товаров** - быстрый доступ к основным категориям
- **Нижнее меню (футер)** - контактная информация, ссылки на социальные сети и дополнительные разделы

### Каталог товаров

В верхнем меню вы можете выбрать одну из следующих категорий:

- **Игры** - видеоигры для различных платформ
- **Консоли** - игровые приставки
- **Аксессуары** - дополнительное оборудование и аксессуары

При выборе категории вы перейдете на страницу с соответствующими товарами.

### Поиск

Для поиска товаров:

1. Введите название товара или ключевые слова в поисковую строку в верхней части сайта.
2. Нажмите кнопку поиска или клавишу Enter.
3. Система отобразит результаты, соответствующие вашему запросу.
4. Используйте фильтры на странице результатов для уточнения поиска.

## Работа с товарами

### Просмотр категорий товаров

На странице категории вы увидите:

- **Список товаров** - карточки товаров с основной информацией
- **Фильтры** - инструменты для фильтрации товаров по различным параметрам
- **Сортировка** - возможность сортировать товары по цене, популярности и т.д.
- **Пагинация** - переключение между страницами результатов

### Просмотр детальной информации о товаре

Для просмотра подробной информации о товаре:

1. Нажмите на название товара или его изображение в списке.
2. На странице товара вы увидите:
   - **Галерея изображений** - фотографии товара с возможностью увеличения
   - **Описание** - подробная информация о товаре
   - **Характеристики** - технические параметры
   - **Цена** - стоимость товара и информация о скидках
   - **Наличие** - информация о доступности товара
   - **Отзывы** - мнения других покупателей
   - **Похожие товары** - рекомендации похожих товаров

### Фильтрация и сортировка

Для удобного поиска товаров используйте:

- **Фильтры** - расположены в левой части страницы категории:
  - Цена (минимальная и максимальная)
  - Производитель
  - Жанр (для игр)
  - Платформа (для игр и аксессуаров)
  - Рейтинг
  - Наличие скидки
  - Другие параметры в зависимости от категории

- **Сортировка** - расположена в верхней части списка товаров:
  - По популярности
  - По цене (возрастание/убывание)
  - По новизне
  - По рейтингу

### Отзывы о товарах

Для просмотра и добавления отзывов:

1. На странице товара перейдите к разделу **"Отзывы"**.
2. Просмотрите существующие отзывы других пользователей.
3. Для добавления своего отзыва (только для авторизованных пользователей):
   - Нажмите кнопку **"Оставить отзыв"**
   - Укажите рейтинг (от 1 до 5 звезд)
   - Напишите текст отзыва
   - Нажмите кнопку **"Отправить"**

## Корзина покупок

### Добавление товаров в корзину

Для добавления товара в корзину:

1. На странице товара нажмите кнопку **"Добавить в корзину"**.
2. При необходимости укажите количество товара.
3. Товар будет добавлен в корзину, и вы увидите соответствующее уведомление.
4. Иконка корзины в верхнем меню обновится, показывая количество товаров.

### Просмотр корзины

Для просмотра содержимого корзины:

1. Нажмите на иконку корзины в верхнем меню.
2. Вы увидите список добавленных товаров с указанием:
   - Изображения товара
   - Названия
   - Цены за единицу
   - Количества
   - Общей стоимости
   - Итоговой суммы заказа

### Изменение количества товаров

Для изменения количества товара в корзине:

1. В корзине найдите товар, количество которого хотите изменить.
2. Используйте кнопки "+" и "-" рядом с полем количества.
3. Или введите нужное число непосредственно в поле.
4. Система автоматически пересчитает стоимость.

### Удаление товаров из корзины

Для удаления товара из корзины:

1. В корзине найдите товар, который хотите удалить.
2. Нажмите кнопку **"Удалить"** рядом с товаром.
3. Товар будет удален из корзины, и итоговая сумма пересчитается.

Для очистки всей корзины нажмите кнопку **"Очистить корзину"**.

## Оформление заказа

### Ввод данных для доставки

Для оформления заказа:

1. В корзине нажмите кнопку **"Оформить заказ"**.
2. Заполните форму с данными для доставки:
   - ФИО получателя
   - Адрес доставки (страна, город, улица, дом, квартира)
   - Контактный телефон
   - Комментарий к заказу (при необходимости)
3. Выберите способ доставки:
   - Курьерская доставка
   - Самовывоз из пункта выдачи
   - Почтовая доставка

### Выбор способа оплаты

После ввода данных для доставки:

1. Выберите предпочтительный способ оплаты:
   - Банковская карта онлайн
   - Электронные платежные системы
   - Наличными при получении
   - Банковский перевод
2. В зависимости от выбранного способа, заполните необходимые данные.

### Подтверждение заказа

Перед окончательным оформлением заказа:

1. Проверьте правильность введенных данных на странице подтверждения.
2. Ознакомьтесь с итоговой суммой заказа, включая стоимость доставки.
3. Нажмите кнопку **"Подтвердить заказ"**.
4. Вы получите подтверждение о успешном оформлении заказа на экране и на указанный email.
5. Система перенаправит вас на страницу с информацией о заказе и дальнейшими инструкциями.

## Личный кабинет

### Просмотр и редактирование профиля

Для управления своим профилем:

1. Войдите в систему и нажмите на свое имя пользователя в верхнем меню.
2. Выберите пункт **"Мой профиль"**.
3. На странице профиля вы можете:
   - Просмотреть и изменить личные данные
   - Изменить пароль
   - Добавить или изменить адреса доставки
   - Настроить предпочтения уведомлений
   - Загрузить аватар

### История заказов

Для просмотра истории заказов:

1. В личном кабинете выберите раздел **"Мои заказы"**.
2. Вы увидите список всех ваших заказов с указанием:
   - Номера заказа
   - Даты оформления
   - Статуса заказа
   - Суммы заказа
3. Нажмите на номер заказа для просмотра подробной информации:
   - Список товаров
   - Данные доставки
   - История статусов
   - Трек-номер (если доступен)

### Избранные товары

Для работы с избранными товарами:

1. Чтобы добавить товар в избранное, нажмите на иконку "сердечка" на карточке товара или на странице товара.
2. Для просмотра избранных товаров перейдите в раздел **"Избранное"** в личном кабинете.
3. Вы можете удалить товар из избранного, нажав на иконку "сердечка" повторно.
4. Из списка избранного вы можете добавлять товары в корзину или переходить на страницы товаров.

### Настройки уведомлений

Для управления уведомлениями:

1. В личном кабинете выберите раздел **"Настройки уведомлений"**.
2. Настройте получение различных типов уведомлений:
   - Статус заказа
   - Акции и специальные предложения
   - Наличие товаров из списка желаний
   - Новости и обновления
3. Выберите предпочтительные способы получения уведомлений (email, SMS).
4. Сохраните настройки, нажав кнопку **"Сохранить"**.

## Дополнительные функции

### Сравнение товаров

Для сравнения товаров:

1. На карточке товара или на странице товара нажмите кнопку **"Добавить к сравнению"**.
2. Добавьте несколько товаров для сравнения (из одной категории).
3. Нажмите на иконку сравнения в верхнем меню или на уведомление о добавлении товара к сравнению.
4. На странице сравнения вы увидите таблицу с характеристиками выбранных товаров.
5. Вы можете удалять товары из сравнения или добавлять их в корзину.

### Список желаний

Список желаний позволяет отслеживать товары, которые вы планируете приобрести в будущем:

1. На странице товара нажмите кнопку **"Добавить в список желаний"**.
2. Для просмотра списка желаний перейдите в соответствующий раздел в личном кабинете.
3. Вы можете создавать несколько списков желаний и давать им названия (например, "День рождения", "Черная пятница").
4. Система будет уведомлять вас о снижении цены на товары из списка желаний (если включено в настройках).

### Подписка на новости

Для получения новостей и специальных предложений:

1. В нижней части сайта (футере) найдите форму подписки на новости.
2. Введите свой email и нажмите кнопку **"Подписаться"**.
3. Подтвердите подписку, перейдя по ссылке в письме, которое придет на указанный email.
4. Управлять подпиской можно в разделе **"Настройки уведомлений"** в личном кабинете.

## Часто задаваемые вопросы (FAQ)

### Доставка и оплата

**Вопрос**: Какие способы доставки доступны?  
**Ответ**: Мы предлагаем курьерскую доставку, самовывоз из пунктов выдачи и почтовую доставку. Конкретные варианты зависят от вашего региона.

**Вопрос**: Сколько стоит доставка?  
**Ответ**: Стоимость доставки зависит от выбранного способа, региона и суммы заказа. При заказе от 5000 рублей доставка может быть бесплатной.

**Вопрос**: Какие способы оплаты доступны?  
**Ответ**: Вы можете оплатить заказ банковской картой онлайн, через электронные платежные системы, наличными при получении или банковским переводом.

### Возврат и гарантия

**Вопрос**: Как вернуть товар?  
**Ответ**: Вы можете вернуть товар в течение 14 дней с момента получения. Товар должен быть в оригинальной упаковке, без следов использования. Для возврата свяжитесь с нашей службой поддержки.

**Вопрос**: Какая гарантия предоставляется на товары?  
**Ответ**: На все товары предоставляется гарантия производителя. Срок гарантии указан в карточке товара и в гарантийном талоне.

### Учетная запись

**Вопрос**: Что делать, если я забыл пароль?  
**Ответ**: На странице входа нажмите ссылку "Забыли пароль?" и следуйте инструкциям для восстановления.

**Вопрос**: Как изменить email в учетной записи?  
**Ответ**: В личном кабинете перейдите в раздел "Мой профиль", внесите изменения и сохраните их.
